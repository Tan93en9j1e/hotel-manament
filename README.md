# HotelManagement

#### 一、接待

负责散客、团体入住登记管理，包含客人姓名、性别、籍贯、工作单位、职业、住店事由、入住日期、预计离店日期、房间号、换房记录等信息登记项目。

#### 二、账务

提供散客、团体单项消费结帐及一次性离店结帐服务，具备预定金管理及杂项消费入帐功能；可查询客人账务情况，在欠账等情况下及时通知客人；结帐时能打印正式收据和明细账单。

#### 三、询问

提供饭店服务指南，供客人查询饭店交通位置、饭店外景、饭店各楼层平面布局、饭店内部客房、餐厅、KTV 包房、会议室的内景等内容；支持查看各餐位灯光效果、各种菜肴的彩色照片等信息。

#### 四、客房

负责客房占用或空闲状态、客房物品种类及状态、客人姓名、客人留言等信息的管理。

#### 五、餐饮

实现对每个餐位点菜、斟酒、入帐及结帐等流程的管理。

#### 六、娱乐

以 KTV 包房为例，提供点歌、乐曲播放、入帐及结帐等功能管理。

#### 七、电话

可与程控交换机联接，实现自动电话计费、自动记入客人账目或结帐、打印正式话费收据等管理功能。

#### 八、人事工资

负责员工姓名、性别、年龄、学历、职务、考勤、工作绩效、工资等信息的管理。

#### 九、财务

根据会计制度设置一级、二级明细科目，汇总入帐；输出总帐、各明细科目帐，生成各类财务报表。

#### 十、库房

用于库存管理，可对每日出入库物品数量及金额进行管理；自动控制物品的库存数量及金额，可根据需要显示或打印各类账目及统计报表（其功能见图 3）。

#### 十一、总经理查询

提供客源、经营状况、内部管理等多方面的查询和统计分析功能，包括客人年构成、饭店收入构成、饭店员工与房间数的配置比例等维度的统计分析。
